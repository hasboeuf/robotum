# Gateway

## Requirements

Python 3.5.2+

## Usage

To run the server, please execute the following from the root directory:

```bash
pip3 install -r requirements.txt
python3 -m auth
```

and open your browser to here:

```bash
http://localhost:8081/v2/ui/
```

## Unittests

```bash
pytest --verbose auth/tests/
```
